🏭 Sumatra Q - Guía Rápida

Sumatra Q es una plataforma web interactiva diseñada para la inspección y el control de calidad de piezas industriales.

✨ Características

  • Visualización 3D: Renderizado de archivos STL.
  • Marcadores: Creación de incidencias táctil.
  • Trazabilidad: Historial inmutable de actualizaciones por usuario y fecha.
  • Filtros Ágiles: Jerarquía visual de incidencias con prioridad.
  • Exportación: Generación de reportes PDF corporativos y listados CSV.

📖 ¿Cómo usar?

  1. Cargar pieza: Usa el botón superior o arrastra un STL a la pantalla.
  2. Crear incidencia: Pulsa el botón (+) y clica en la pieza 3D.
  3. Filtrar: Usa los botones superiores para aislar los problemas urgentes.
  4. Reporte: Filtra la vista deseada y pulsa "Generar PDF".
Desarrollado para la mejora continua del Control de Calidad.
